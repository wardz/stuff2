How to install addons

Updated 10.07.2016

Prerequisites

If you have the World of Warcraft client running, exit it completely.


Be aware that some vanilla addons may only work with the English game client.

Extract/unzip

All downloaded files are zipped (compressed) to save bandwidth. Therefore you will need to unzip the addon before installing it. You can unzip a file by right clicking on it and selecting "Extract here", or by double clicking the file and then drag & drop the folders you see in the new window over to e.g your desktop.

This process may work differently depending on what kind of zip application you are using. Read the manual for your app if you cannot get it to work.


Install

Find out where your game client is located on your computer. (Where you run World of Warcraft from).

  • Open your World of Warcraft folder.
  • Go to "Interface" folder.
  • Go to "AddOns" folder.
  • Drag/copy all of the files you extracted earlier into this location. (Not the .zip file!)
If you cannot find the Interface folder and you have started the game atleast once, it means you are in the wrong location. You are most likely browsing the folder for the shortcut to World of Warcraft.


Enable out of date addons

Login to your World of Warcraft client. At the character selection screen there should be a button in the lower left corner saying "Addons", click it.
In the new window, check the checkbox found in the top right corner saying "Enable out of date addons". Click "Okay" button to save.

If you cannot find the addon button, it means you did not install the addon(s) properly. If you do find it, you are done installing the addon(s).


Important

If you are installing only textures and not an addon, you need to drag the extracted files into the "Interface" folder, instead of the "AddOns" folder.

If you get access denied errors when copying the addons, easiest solution is to install World of Warcraft outside "Program Files" folder on your computer.

If you do not have permission issues but still cannot get the addons to work I recommend trying the above anyways.
Because sometimes modifications in "Program Files" folder does not actually get saved in that location, but instead gets redirected to a different folder called Virtual User Store, where World of Warcraft will not look for your addons.
